##How to run this app
---------------------
* Brew Install ImageMagick
`````````
$ brew install ImageMagick
`````````
* Bundle install gems
`````````
$ bundle install
`````````
* Create the data Base
``````
$ rake db:create
``````
* Then migrate the tables
```````
$ rake db:migrate
```````
* launch the server
````
$ rails s
````
